## About

I work where BIM, geospatial data, and code overlap. Most of what I do sits between Revit and the formats downstream systems actually consume — CityGML, IMDF, 3D Tiles, FBX, glTF. I write Revit plugins in C#, Python converters with GeoPandas and Shapely, and FME workbenches that replace legacy multi-step exports.

Before development I spent seven years modeling hospitals, housing, and an airport terminal expansion at Tengbom in Stockholm. That background still informs the work — I care about what a valid BIM model actually looks like, not just what the schema says it should.

---

## What I'm working on

- Revit plugins and Python tools for BIM-to-geospatial workflows
- Indoor navigation data pipelines (IMDF, CityGML, 3D Tiles)
- PLATEAU integration for Japanese station areas
- Reducing the number of file format hops between Revit and whatever eventually renders the data

## Selected Work

### Shinjuku and surrounding buildings
Revit models of the buildings around Shinjuku Station — Lumine 1, Lumine EST, NEWoMan, Busta, and Yodobashi East and West. Output goes two places: FBX for the Unity-based indoor navigation app (Sta.navi), and CityGML and 3D Tiles for public release under PLATEAU.

[View project](https://malmqvist.dev/en/projects/shinjuku-nav)

### Ersta Sjukhus
Rebuilt the Revit model from scratch based on an IFC export from the previous architect. Primary modeler for the anodized aluminum facade. Won Best Healthcare at the Monocle Design Awards 2024.

[View project](https://malmqvist.dev/en/projects/ersta-sjukhus)

### Revit → GeoPackage export plugin
C# / Revit API plugin that exports directly to GeoPackage with an IMDF category preview, EPSG selection, and multi-floor support. Replaced an older Revit → DWG → Shapefile pipeline at work.

[View project](https://malmqvist.dev/en/projects/revit-geopackage)
